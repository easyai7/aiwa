さばきが延期された理由
ローマ人への手紙 2:1–5
まず直視すべき問い
悪人がすぐに罰を受けないのは
なぜか？
神を離れても幸せそうに見える人がいる。
罪を犯しても何も起きない。
この「延期」には、神の深い意図がある。
しかし — 聖書が宣言する真実
延期は神の憐れみであり
悔い改めへの招きである
「神の慈愛があなたを悔い改めに導く」（ローマ 2:4）
この時間は、あなたへの最後の招きかもしれない。
⚠ さばきの順序を整理する
一般的な理解：罪 → 神の怒り → さばき → 苦しみ
ローマ書の説明：神への反逆 → さばき（放置）→ 罪 → 汚れ → 悲惨

苦しみはさばきの結果である場合と、訓練の過程である場合がある。 苦しみの有無がさばきの基準ではなく、神との関係が繋がっているかどうかが問題。
神を離れた状態 — それ自体がさばき
ローマ書1章のさばきとは、神を離れた状態に放置されることです。その状態で行うすべての行いは、神への怒りを積み上げるものでしかありません。善人に見える行いも、神の前では罪として現れます。家出した子が友人に親切にしても、親への不孝は変わらないように。
ローマ 1:28 / ローマ 2:5
自分で自分を定罪している
ローマ 2:1は「判断するな」という意味ではありません。他人の罪を判断しながら、自分も同じことをしているなら、その判断で自分を定罪していることになります。人は他人の罪はよく見えるが、自分の罪は見えにくい。気づかないからといって罪がないわけではありません。
ローマ 2:1
死刑囚の例え — 執行猶予の意味
死刑判決が出ても、すぐに執行されるわけではありません。死刑囚も拘置所の中で友人と笑い、食事を楽しみます。しかし処刑の日は近づいています。神を離れた人の一時的な幸福もこれと同じです。最終さばきは必ず来ます。この延期は神の憐れみによるものです。
ローマ 2:4 / マタイ 5:46
権士女の臨終 — 神との約束が果たされるまで
伝道師時代、意識のない権士女が何日もなかなか息を引き取られなかった。未払いの献金が判明し、息子がそれを納めると約束した朝、権士女は静かに召された。神との約束が果たされるまで、神は命を延ばしてくださった。私たちが今も生きている理由も、神の前でまだ果たすべき使命があるからかもしれない。
エゼキエル 3:18
今生きている — それは使命がある証拠
不信者に長い時間が与えられているのは、悔い改めの十分な機会のためです。その時間は私たちにとって伝道の機会です。その長い期間に一度も福音を伝えなかったなら、私たちもさばきを免れないでしょう。あなたが今もここにいる理由は、まだ果たすべき使命があるからです。
エゼキエル 3:18 / ローマ 2:4
月曜日
苦しみの二つの意味を考える
苦しみには二種類あります。罪の結果としての苦しみと、神の訓練としての苦しみ。今あなたが経験している苦しみはどちらですか？神との関係の中にある苦しみには、希望と慰めがあります。今日、その苦しみを神に持っていってください。
ローマ 2:4 —「神の慈愛があなたを悔い改めに導く」
火曜日
神を離れた行いはすべて罪
神を離れた状態では、どんなに善く見える行いも神の前には罪として現れます。今日、自分の行いの動機を点検してください。神のためですか、それとも人のためですか、自分のためですか。神との関係の中から行動が生まれるよう、今日祈ってください。
ローマ 2:5 —「自分への怒りを積み上げている」
水曜日
他人を裁く前に自分を見る
他人の罪はよく見えるのに、自分の罪は見えにくい。今日、誰かを心の中で批判したとき、立ち止まってみてください。自分も同じことをしていないか。気づかないからといって罪がないわけではありません。今日、自分自身に正直になる時間を持ちましょう。
ローマ 2:1 —「あなたも同じことを行っている」
木曜日
延期は憐れみ — 悔い改めへの招き
悪人がすぐに罰を受けないのは、神の憐れみです。その時間は悔い改めへの招きです。今日、まだ神を知らない周りの人を思い浮かべてください。神はその人にも、今日も憐れみを持って待っておられます。あなたはその人に伝えましたか？
ローマ 2:4 —「豊かな慈愛と忍耐と寛容」
金曜日
権士女の話から学ぶ
神との約束が果たされるまで命が延ばされた権士女の話を思い出してください。私たちも今ここに生きているのは、まだ神の前で果たすべきことが残っているからかもしれません。今日、神の前にまだ果たしていないことは何か、静かに考えてください。
エゼキエル 3:18 —「悪人に警告せよ」
土曜日
与えられた時間を無駄にしない
長い人生という時間は、回り道ではありません。神が悔い改めと伝道のために与えてくださった時間です。この時間を自分の成功や幸福だけのために使い果たすなら、その時間そのものがさばきの証拠になります。今週残りの時間をどう使いますか？
ローマ 2:5 —「怒りの日のために怒りを積み上げている」
礼拝準備
今週を振り返りながら
神の憐れみによって延ばされたこの時間を、あなたはどう使いましたか？自分の罪を認めましたか？誰かに伝えましたか？今日の礼拝は義務ではなく、生かされているこの命への感謝から始めましょう。
詩篇 100:4 —「感謝しながら、その門に入れ」
分かち合いの中心テーマ
神はなぜ悪人をすぐに滅ぼされないのか。
この延期の意味を正しく理解し
与えられた時間をどう生きるかを考えましょう。
1
「苦しみはさばきの証拠」という考えと「苦しみは訓練の過程」という考え。あなたはどちらに偏っていましたか？この二つをどう区別できますか？
分かち合いの方向 → 苦しみの中で神との関係が繋がっているかどうかが鍵。苦しみそのものではなく、神との関係の中にいるかどうかを確認しましょう。
2
「神を離れた状態では善い行いも罪になる」という説明は、あなたにどのように響きましたか？家出した子の例え、海賊の例えを思い出しながら話し合ってください。
分かち合いの方向 → 行いの善悪より、神との関係が先。これは不信者への伝道の緊急性にも繋がります。
3
権士女の臨終の話を聞いて、何を感じましたか？あなたが今もここに生きている理由は何だと思いますか？
分かち合いの方向 → 具体的に。神の前でまだ果たすべき使命を、一人ひとりが言葉にしてみましょう。
4
あなたの周りで、神の憐れみの時間がまだ与えられている人は誰ですか？その人のために、今週具体的に何ができますか？
分かち合いの方向 → 名前を挙げて。祈りで終わりましょう。
